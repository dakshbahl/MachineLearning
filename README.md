# Machine Learning Projects

This repository contains my Python and machine learning projects, including work with classical machine learning models, deep learning concepts, and financial data analysis.

The goal of this repository is to document my growth in machine learning and showcase the projects, models, and experiments I have worked on while studying computer science and contributing to the NYU Stern Business Analytics Club Machine Learning Team.

## Overview

This repo includes a collection of machine learning projects focused on data preprocessing, model training, prediction, classification, and analysis. The projects cover both foundational ML techniques and more advanced model architectures.

Some of the main areas explored include:

* Random Forest models
* Transformer-based models
* Classification problems
* Financial anomaly detection
* Data preprocessing pipelines
* Model evaluation
* Python-based machine learning workflows

## Projects

### Random Forest Models

This section includes projects using Random Forest models for classification and prediction tasks. Random Forests are useful for working with structured data, identifying feature importance, and building strong baseline models.

Topics covered include:

* Training Random Forest classifiers
* Evaluating model accuracy
* Comparing model performance
* Understanding feature importance
* Applying models to financial datasets

### Transformer Models

This repository also includes work related to Transformer models, which are commonly used for sequence-based tasks and modern machine learning applications.

Topics covered include:

* Understanding Transformer architecture
* Applying attention-based models
* Working with sequence data
* Exploring how Transformers can be used for pattern recognition and anomaly detection

### Financial Machine Learning

Some projects focus on applying machine learning to financial data. This includes exploring how models can be used to detect unusual patterns, classify behavior, and support data-driven decision making.

Topics covered include:

* Insider-trading detection concepts
* Financial anomaly detection
* Market-related data analysis
* Model comparison between classical ML and deep learning methods
